## Usage

Create `/config/firebase.js` with:

```javascript
module.exports = {
  apiKey: "???",
  authDomain: "???",
  databaseURL: "???",
  projectId: "???",
  storageBucket: "???",
  messagingSenderId: "???"
};
```

Create Firebase Instance on Google Firebase Console and enable Google Login. Then you can have fun!
